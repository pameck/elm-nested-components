# elm-nested-components

Example of an elm single page app that renders independent modules.

Pre-requisites:

* `npm`

## Install it

1. Install `elm`

  `npm install -g elm`

2. Install the project dependencies

  `npm i`

## Run it

There are two components to run separately:

1. Run the api

  `npm run api`

1. Run the client (*on a different terminal instance*):

  `npm run client`
